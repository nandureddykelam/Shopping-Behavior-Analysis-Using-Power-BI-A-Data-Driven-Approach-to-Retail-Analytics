# Sales Analysis Dashboard using Power BI

## 📌 Project Overview
This project is a **Sales Analysis Dashboard** created using **Power BI** to analyze sales performance, revenue trends, profit margins, and customer behavior.  
The dashboard provides interactive visualizations that help in making data-driven business decisions.

---

## 🎯 Objectives
- Analyze total sales and profit performance
- Identify top-selling products
- Compare regional sales performance
- Track monthly and yearly sales trends
- Understand customer purchasing patterns

---

## 🛠️ Tools & Technologies Used
- **Power BI**
- **Microsoft Excel / CSV Dataset**
- **DAX (Data Analysis Expressions)**
- **Power Query**

---

## 📊 Dashboard Features
- KPI Cards (Total Sales, Profit, Orders, Customers)
- Sales Trend Analysis
- Region-wise Sales Comparison
- Category & Product Performance
- Monthly Revenue Growth
- Interactive Filters / Slicers

---

## 📂 Files Included
- `Sales_Analysis.pbix` → Power BI project file  
- `sales_data.xlsx` / `sales_data.csv` → Dataset used  
- `screenshots/` → Dashboard preview images  
- `README.md` → Project documentation  

---

## 📈 Insights Generated
- Top-performing products generate maximum revenue
- Certain regions have higher sales growth
- Seasonal trends affect monthly sales
- Profit margins vary across categories
